Now you may run NymphCast Server from the Start menu:

- NymphCastServer > NymphCast Server - Audio
- NymphCastServer > NymphCast Server - Video
- NymphCastServer > NymphCast Server - GUI
- NymphCastServer > NymphCast Server - Screen saver

Or you can run NymphCast Server with a configuration file from a console:

NymphCastServer.exe -c /path/to/config.ini

Configuration files are provided in the `config` subdirectory of the installation directory.
